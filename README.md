# Prime Number Generator

This repository contains a simple Python script that generates prime numbers up to a user-specified limit using the Sieve of Eratosthenes algorithm.

## Requirements

- Python 3.7 or higher

## Usage

Run the script from the command line:

```bash
python3 prime_generator.py --limit 100
```

Replace `100` with any positive integer to control the upper bound for prime search. The script will output a list of all primes up to that number.

## Example

Running `python3 prime_generator.py --limit 20` produces:

```
[2, 3, 5, 7, 11, 13, 17, 19]
```
