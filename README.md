# Rock-Paper-Scissors-Lizard-Spock

This repository contains an expanded command-line game inspired by "Rock Paper Scissors Lizard Spock." It features colored output, a running score, and configurable match length.

## Requirements

- Python 3.8+
- `colorama` (install with `pip install colorama`)

## Running the Game

```bash
python game.py
```

Follow the on-screen prompts to choose between `rock`, `paper`, `scissors`, `lizard`, or `spock`. The first player to reach the target number of wins takes the match. Type `quit` at any time to exit.
